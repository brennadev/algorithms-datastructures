Readme - Priority Queue

----------------------------------------------------

PriorityQueue: An object-oriented priority queue
    PriorityQueue's heap is implemented using a vector, so there's no arbitrary max size

----------------------------------------------------

To run: just use run.sh with a txt file in the current directory with the appropriate format

Input file's format must be as follows:
name1 : priority1
name2 : priority2
name3 : priority3
